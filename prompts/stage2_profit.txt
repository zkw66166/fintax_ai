【利润表计算逻辑（用于理解数据关系，不需要在SQL中计算）】
- 营业利润 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 研发费用 - 财务费用 + 其他收益 + 投资收益 + 公允价值变动收益 - 信用减值损失 - 资产减值损失 + 资产处置收益
- 利润总额 = 营业利润 + 营业外收入 - 营业外支出
- 净利润 = 利润总额 - 所得税费用
- 综合收益总额 = 净利润 + 其他综合收益的税后净额

【结构分析/构成分析模式（UNPIVOT）】★ 当用户问"结构"/"构成"/"组成"/"各项占比"时必须使用此模式 ★
当用户问利润表项目的结构组成时（如"期间费用构成"/"利润总额构成"），
需将宽表的多个组件列拆为行，每行一个组件，包含金额和占比。
使用 UNION ALL 将各组件列拆为 (项目名称, 金额, "占比(%)") 行。

⚠️ 重要：构成分析查询必须返回以下列：
- period_year, period_month（如果有多个期间）
- "项目名称"（字符串列）
- "金额"（数值列）
- "占比(%)"（百分比列）

示例SQL（单期间期间费用构成分析）：
WITH latest AS (
  SELECT taxpayer_id, period_year, period_month, time_range, revision_no,
         selling_expense, administrative_expense, rd_expense, financial_expense,
         ROW_NUMBER() OVER (PARTITION BY taxpayer_id, period_year, period_month, time_range ORDER BY revision_no DESC) AS rn
  FROM vw_profit_eas
  WHERE taxpayer_id = :taxpayer_id AND period_year = :year AND period_month = :month AND time_range = '本年累计'
),
base AS (SELECT * FROM latest WHERE rn = 1),
total AS (SELECT (selling_expense + administrative_expense + rd_expense + financial_expense) AS period_expense_total FROM base),
items AS (
  SELECT '销售费用' AS "项目名称", selling_expense AS "金额",
         ROUND(selling_expense * 100.0 / NULLIF((SELECT period_expense_total FROM total), 0), 2) AS "占比(%)" FROM base
  UNION ALL
  SELECT '管理费用', administrative_expense,
         ROUND(administrative_expense * 100.0 / NULLIF((SELECT period_expense_total FROM total), 0), 2) FROM base
  UNION ALL
  SELECT '研发费用', rd_expense,
         ROUND(rd_expense * 100.0 / NULLIF((SELECT period_expense_total FROM total), 0), 2) FROM base
  UNION ALL
  SELECT '财务费用', financial_expense,
         ROUND(financial_expense * 100.0 / NULLIF((SELECT period_expense_total FROM total), 0), 2) FROM base
)
SELECT "项目名称", "金额", "占比(%)" FROM items WHERE "金额" > 0
ORDER BY "金额" DESC
LIMIT {max_rows}
