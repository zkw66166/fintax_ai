【修订版本处理】
若需取最新修订版本，使用窗口函数：
WITH latest AS (
  SELECT taxpayer_id, taxpayer_name, period_year, period_month,
         account_code, account_name, level, category, balance_direction,
         revision_no,
         <需要查询的指标列>,
    ROW_NUMBER() OVER (
      PARTITION BY taxpayer_id, period_year, period_month, account_code
      ORDER BY revision_no DESC
    ) AS rn
  FROM vw_account_balance
  WHERE taxpayer_id = :taxpayer_id AND period_year = :year AND period_month = :month
)
SELECT <columns> FROM latest WHERE rn = 1

【科目过滤】
- 按科目名称过滤：account_name = '库存现金'
- 按科目类别过滤：category = '资产'（可选值：资产/负债/权益/成本/损益）

  例如：account_name LIKE '应交税费%增值税%'  → 查询应交税费下的增值税明细

- 多个子科目汇总：可用 OR 或 IN 组合多个 LIKE 条件
