# 智能财税咨询系统-企业画像可视化分析页面 详细设计描述
本页面为**企业级SAAS智能财税咨询系统**的核心功能页——企业画像可视化分析，面向企业用户，整体为**电脑端工作台布局**，风格是专业商务科技风，以蓝白为主色调、浅灰为分隔色、暖橙/深灰为高亮色，采用**卡片式分模块+网格布局**，信息层级分明，数据展示结合文字标值、轻量可视化图表、资质标签、评级标注，无冗余元素，符合财税行业严谨、专业的视觉与内容要求，页面适配1920*1080主流电脑分辨率，支持模块折叠/展开、数据hovertooltip、核心数值跳转详情页等基础交互。
画像截图中的指标值是当前企业示例值。

## 五、视觉与交互统一规范
### 1. 色彩规范
- 主色：#1677ff（商务蓝，用于选中项、企业名称、核心标题）；
- 背景色：#ffffff（纯白，页面、导航栏、卡片底色）；
- 分隔色：#f5f7fa（浅灰，模块分隔线、卡片边框）；
- 高亮色：#ff7d00（暖橙，评级标签、优秀/良好备注）；
- 文字色：#333333（黑色，主标题）、#666666（深灰，子标题）、#999999（浅灰，备注、时间、版权）；
- 警示色：#f53f3f（红色，本页面无风险数据，预留用于风险预警）。

### 2. 字体规范
- 主标题（企业名称、模块标题）：思源黑体/微软雅黑 Bold，16-20px；
- 子标题（模块内分类）：思源黑体/微软雅黑 Bold，14px；
- 正文（指标名称、备注）：思源黑体/微软雅黑 常规，12-14px；
- 核心数据（数值、占比）：思源黑体/微软雅黑 SemiBold，比正文大1号；
- 辅助文字（时间、版权、无数据）：思源黑体/微软雅黑 常规，12px，浅灰色。

### 3. 交互规范
1. 模块交互：所有十四大模块支持**折叠/展开**，折叠后仅显示模块标题，展开后展示全部内容；
2. 数据交互：核心数值（如营收、纳税、研发投入）**可点击跳转**至对应数据详情页；所有数据项**hover显示tooltip**，补充数据定义、统计口径、更新时间；
3. 导航交互：左侧导航栏项**hover变蓝**，点击后标亮并跳转对应页面；顶栏功能标签同理；
4. 适配交互：页面支持**1920*1080/1366*768**主流分辨率自适应，模块卡片自动调整宽度，可视化图表等比缩放。

### 4. 可视化图表规范
所有图表均为**轻量极简风格**，无多余装饰，仅保留坐标轴、数值标注、图例，与卡片内文字信息对齐，配色遵循页面主色规范：
- 饼图/环形饼图：股权/人员/学历构成，用4种浅色系区分，标注股东/类别名称+占比；
- 柱状图：税种构成/资产负债，仅保留纵轴刻度与数值柱，标注具体金额；
- 折线图：专利趋势，仅保留横轴年份、纵轴刻度、趋势线，标注各年份专利数量；
- 进度条：财务能力评分，蓝色填充进度，标注百分比，未填充部分为浅灰色。
