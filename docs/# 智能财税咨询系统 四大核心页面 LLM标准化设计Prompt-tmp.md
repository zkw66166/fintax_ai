# 智能财税咨询系统 四大核心页面 LLM标准化设计Prompt

## 设计核心要求
生成**创智软件股份有限公司智能财税咨询系统**的4个核心页面：单户企业数据、多户企业数据、数据浏览（通表格式）、数据浏览（原表格式），整体为企业级后台管理系统风格，严格遵循**左侧固定导航栏+右侧自适应主内容区**布局，一屏展示所有内容无滚动条/分页；配色仅用白（底色）、黑（主文字/加粗）、灰（辅助文字/边框/占位符）三色，无任何彩色标识；字体统一为无衬线字体，层级为「模块标题/表格表头（加粗）> 操作区文字/表格内容（常规）> 底部辅助信息（灰色小号）」；所有交互元素（选择器、按钮、视图切换）均为**纯文字链接式**，无背景/边框/圆角，选中态无特殊标色，仅以页面内容对应；表格均为浅灰色单边框，表头白色背景无隔行变色，文字/日期列左对齐、数字/金额列右对齐；保留所有页面中的占位符（N、Q、=、S）原位置，无实际图标设计；数据格式统一：日期为`YYYY-MM-DD`、期间为`YYYY年第X季度`，金额按视图差异化展示。

## 全局通用元素设计规范
### 1. 顶部通栏（所有页面统一，仅细微信息调整）
- 布局：页面最上方通栏，贯穿全宽，纯白色无分割线，内容左中右分布
- 左侧：首行中文「智能财税咨询系统」，次行大号加粗英文「ENTERPRISE FINANCIAL & TAX INTELLIGENCE PLATFORM」，第三行中号常规字体「创智软件股份有限公司」
- 右侧：时间戳「2026/02/21 17:03:36」+ 身份标识「超级管理员XX + 灰色辅助文字企业用户」（XX按页面调整：单户=Jo、多户=Jo、通表=Jo、原表=D0）
### 2. 左侧导航栏（固定宽度，垂直排列，浅灰竖线与主内容分隔）
- 导航项（基础）：工作台/自工作台、单户企业数据、多户企业数据、数据浏览、AI智问、企业画像、数据管理、系统设置
- 选中态：对应页面模块无特殊标色，仅主内容区匹配
- 排版/占位符：多户页面「数据浏览」左侧加Q占位符、「企业画像/数据管理/系统设置」横向紧凑排列；其余页面导航项均垂直单独排列
- 笔误：通表页面保留「自工作台」，单户/多户/原表页面为「工作台」
### 3. 底部全局信息（所有页面统一，仅版本号/语言标识调整）
- 布局：主内容区最底部，左中右分布，纯白色底色
- 左侧：灰色小号字体「2024智能财税咨询系统All rights reserved」
- 右侧：灰色小号字体版本号（单户=v1.0、多户=v1.0.0、通表=v1.0.、原表=v1.0. S）+ 语言标识「中」（仅单户、原表页面展示）
### 4. 通用表格规范
- 边框：浅灰色单边框，表头白色背景，无隔行变色
- 对齐：文字/日期/期间列左对齐，数字/金额/ID列右对齐
- 字体：表头黑色加粗，内容黑色常规
- 数据格式：日期`YYYY-MM-DD`、期间`YYYY年第X季度`，金额无货币符号
