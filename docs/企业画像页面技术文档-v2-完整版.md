# 企业画像页面 V2 技术文档（完整版）

## 1. 系统架构总览

### 1.1 技术栈

| 层级 | 技术 | 版本 |
|------|------|------|
| 前端框架 | React + Vite | 19.2.0 / 7.2.4 |
| 图表库 | Chart.js | 4.5.1 |
| 图标库 | Lucide React | 0.562.0 |
| 路由 | React Router DOM | 7.13.0 |
| 后端框架 | FastAPI + Uvicorn | 0.115.0 |
| 数据库 | SQLite | 3.x |
| 定时任务 | APScheduler | 3.11.2 |

### 1.2 文件清单

| 文件 | 路径 | 行数 | 职责 |
|------|------|------|------|
| CompanyProfile.jsx | `frontend/src/components/` | 989 | 前端主组件 |
| CompanyProfile.css | `frontend/src/components/` | ~1190 | 样式表 |
| company_profile.py | `server/routers/` | 1641 | 后端 API 路由 |
| api.js | `frontend/src/services/` | 418 | 前端 API 客户端 |
| main.py | `server/` | 88 | FastAPI 入口 |
| create_profile_tables_v2.py | `database/` | 397 | 数据库迁移脚本 |

### 1.3 数据流

```
用户选择企业 + 年份
       ↓
CompanyProfile.jsx useEffect 触发
       ↓
fetch(`/api/company-profile/${companyId}/full?year=${year}`)
       ↓
Vite dev proxy → http://localhost:8000
       ↓
FastAPI: get_full_company_profile()
  → 并行调用 21 个子查询函数
  → 每个函数独立查询 SQLite (financial.db)
       ↓
返回聚合 JSON（约 20 个顶层字段）
       ↓
React setState(profile) → 渲染 14 个 profile-card
```

## 2. 页面顶部：Header 区域

### 2.1 结构

```
┌─────────────────────────────────────────────────────────────┐
│  企业画像可视化分析    │  企业名称 + 信用代码  │  年份选择 + 刷新按钮  │
└─────────────────────────────────────────────────────────────┘
```

### 2.2 元素明细

| 元素 | 组件/标签 | 数据来源 | CSS 类 |
|------|-----------|----------|--------|
| 页面标题 | `<h1>企业画像可视化分析</h1>` | 静态文本 | `.profile-header h1` |
| 企业名称标签 | `<span className="company-tag">` | `companies.find(c => c.id === companyId)?.name` | `.company-tag` |
| 统一社会信用代码 | `<span className="credit-code">` | `basic_info.credit_code` | `.credit-code` |
| 年份选择器 | `<select className="year-selector">` | 固定选项: 2024/2023/2022 | `.year-selector` |
| 刷新指标按钮 | `<button className="recalculate-btn">` | 点击 POST `/api/admin/metrics/recalculate` | `.recalculate-btn` |

### 2.3 交互逻辑

- 年份切换：`onChange` → `setYear(Number(e.target.value))` → 触发 `useEffect` 重新加载
- 刷新指标：POST 请求发送 `{company_ids: [companyId], year}` → 5 秒后 `window.location.reload()`
- Header 使用 `position: sticky; top: 0; z-index: 10` 固定在顶部

### 2.4 状态页面

| 状态 | 条件 | 显示 |
|------|------|------|
| 加载中 | `loading === true` | 旋转动画 + "加载企业画像..." |
| 错误 | `error !== null` | ⚠️ 图标 + 错误信息 |
| 空状态 | `profile === null` | 📊 图标 + "请选择企业查看画像" |

## 3. 辅助组件体系

### 3.1 EvalLabel（评价标签）

```jsx
<EvalLabel text="正常" type="positive" />
```

| type | CSS 类 | 背景色 | 文字色 | 语义 |
|------|--------|--------|--------|------|
| positive / growth | `eval-green` | #dcfce7 | #166534 | 正面/增长 |
| neutral | `eval-blue` | #dbeafe | #1e40af | 中性 |
| warning | `eval-yellow` | #fef9c3 | #a16207 | 警告 |
| negative | `eval-red` | #fee2e2 | #dc2626 | 负面 |
| purple | `eval-purple` | #f3e8ff | #7c3aed | 特殊标记 |
| orange | `eval-orange` | #ffedd5 | #c2410c | 辅助标记 |

样式：`display: inline-block; padding: 2px 8px; border-radius: 10px; font-size: 11px`

### 3.2 SectionTitle（二级模块标题）

```jsx
<SectionTitle name="基本工商信息" color="green" />
```

结构：圆点（8×8px 圆形）+ 标题文字（13px, #475569）
圆点颜色：green(#10b981) / purple(#8b5cf6) / orange(#f59e0b) / blue(#3b82f6) / cyan(#06b6d4)

### 3.3 CompactMetric（紧凑指标行）

```jsx
<CompactMetric label="注册资本" value="500万元" evalInfo={{text:'充足', type:'positive'}} bgColor="bg-purple" />
```

布局：`display: flex; justify-content: space-between`，左侧 label（#64748b），右侧 value（#1e293b, font-weight:500）+ 可选 EvalLabel

背景色变体：`bg-purple`(#faf5ff) / `bg-orange`(#fff7ed) / `bg-blue`(#eff6ff) / `bg-cyan`(#ecfeff) / `bg-green`(#f0fdf4) / `bg-white`(#ffffff)

### 3.4 ProgressBar（进度条）

```jsx
<ProgressBar label="盈利能力" value={75} max={100} color="blue" />
```

结构：标题行（label + value%）+ 进度条（8px 高，圆角 4px）
颜色：blue(#3b82f6→#60a5fa) / purple(#8b5cf6→#a78bfa) / green(#10b981→#34d399) / orange(#f59e0b→#fbbf24)
动画：`transition: width 0.3s ease`

### 3.5 图表组件

#### PieChart（饼图）
- 基于 Chart.js `type: 'pie'`
- 图例位置：bottom，字号 11px
- 边框：2px 白色
- 使用 `useRef` + `useEffect` 管理 Chart 实例生命周期

#### BarChart（柱状图）
- 基于 Chart.js `type: 'bar'`
- 圆角：`borderRadius: 4`
- Y 轴从 0 开始，网格色 #f3f4f6
- 隐藏图例

#### LineChart（折线图）
- 基于 Chart.js `type: 'line'`
- 支持多系列（`lines` 数组）
- `tension: 0.3`（平滑曲线），`fill: true`（面积填充，20% 透明度）

### 3.6 OrgTree（组织架构树）

```jsx
<OrgTree rootName="企业名称" children={[{name:'股东A', value:60}, ...]} />
```

结构：根节点（蓝色背景 #eff6ff）→ 垂直连接线（15px）→ 水平连接线 → 子节点列表
子节点名称截断：`substring(0, 8)` + 省略号
CSS 连接线通过 `::before`（垂直线）和 `::after`（水平线）伪元素实现

| <0% | 下降 / red |

|--------|------|
