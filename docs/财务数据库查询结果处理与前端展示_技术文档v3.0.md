## 一、系统架构概述

财务数据查询系统采用前后端分离架构，通过 SSE（Server-Sent Events）实现实时流式数据传输，支持文本内容与图表数据的混合推送。

### 1.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 | FastAPI + Python | RESTful API 服务 |
| 数据库 | SQLite | financial.db 企业财务数据库 |
| 前端 | React + Chart.js | UI 渲染与图表展示 |
| 实时通信 | SSE (Server-Sent Events) | 服务端推送 |

### 1.2 数据流架构

```
用户问题
    │
    ▼
┌─────────────────────────────────────────────────┐
│  FastAPI /chat/stream 端点                      │
│  (server/routers/chat.py)                      │
└─────────────────────────────────────────────────┘
    │
    ├──▶ FinancialQuery.search() ──▶ financial.db
    │       │
    │       ├── 格式化方法: format_results()
    │       │   └── 返回 Markdown 表格/列表
    │       │
    │       └── 图表数据生成: 内存中构建 chartData
    │           └── yield send_event("chart", chartData)
    │
    ├──▶ SSE Event: "message" ──▶ 前端渲染文本
    │
    └──▶ SSE Event: "chart" ──▶ 前端 ChartRenderer
            │
            └── 图表组件渲染
```

```python
def search(self, question: str) -> Tuple[Optional[List[Dict]], Optional[Dict], str]:
    """
    主查询入口
    """
    # 1. 识别企业名称
    company = self.match_company(question)
    if not company:
        return None, None, "company_not_found"
    
    # 2. 提取时间范围
    time_range = self.extract_time_range(question)
    
    # 3. 识别指标
    metrics = self.extract_metrics(question)
    
    # 4. 执行查询 (Text-to-SQL)
    results = self.execute_query(company['id'], time_range, metrics, question)
    
    if not results:
        return None, company, "no_data"
    
    return results, company, "success"
```

```python
def format_results(self, results: List[Dict], company: Dict) -> str:
    """
    格式化查询结果
    当指标 > 1 或期间 >= 4 时使用表格格式
    """
    # 统计不同指标和期间
    metrics = set(r['metric_name'] for r in results)
    periods = set((r['year'], r.get('quarter')) for r in results)
    
    # 判断是否使用表格格式
    use_table = len(metrics) > 1 or len(periods) >= 4
    
    if use_table:
        return self._format_as_table(results, metrics, periods)
    else:
        return self._format_as_list(results)
```

```python
def _format_value(self, value, unit: str) -> str:
    """格式化数值 - 自动单位转换"""
    if value is None:
        return "暂无数据"
    
    if unit == '元':
        if abs(value) >= 100000000:
            return f"{value/100000000:.2f}亿"
        elif abs(value) >= 10000:
            return f"{value/10000:.2f}万"
        else:
            return f"{value:.2f}"
    else:
        return f"{value:.2f}{unit}"

def _format_change(self, change, unit: str) -> str:
    """格式化增长额（带正负号）"""
    if unit == '元':
        if abs(change) >= 100000000:
            return f"{change/100000000:+.2f}亿"
        elif abs(change) >= 10000:
            return f"{change/10000:+.2f}万"
        else:
            return f"{change:+.2f}"
    else:
        return f"{change:+.2f}{unit}"
```

### 5.1 模式概述

| 模式 | 参数值 | 展示内容 |

```javascript
    const formattedDatasets = datasets.map((ds, idx) => ({
        ...ds,
        backgroundColor: COLORS[idx % COLORS.length],
        borderColor: BORDER_COLORS[idx % COLORS.length],
        borderWidth: 1,
        borderRadius: 4,
        fill: chartType === 'line' ? 'origin' : undefined
    }));
    
    // 渲染图表
    return (
        <div className="chart-container">
            {chartType === 'bar' && <Bar data={{ labels, datasets: formattedDatasets }} options={options} />}
            {chartType === 'combo' && <Bar data={{ labels, datasets: formattedDatasets }} options={options} />}
        </div>
    );
}
```

## 附录 D: 比率指标特殊处理机制

### D.1 概述

财务指标中的比率类指标（如税率、利润率、资产负债率等）具有特殊性：它们通常存储的是**期间值**（如季度值），当用户询问"全年"的比率时，需要进行**聚合计算**（平均值/最大/最小值），而不是简单累加。

系统针对比率指标实现了以下特殊处理机制：

### D.2 单位识别

文件: `modules/financial_query.py`，行号 998-1009

系统通过指标名称自动识别单位类型：

```python
def _get_unit(self, metric_name: str) -> str:
    """获取指标单位"""
    if '率' in metric_name or 'ROA' in metric_name or 'ROE' in metric_name:
        return '%'
    elif '天数' in metric_name:
        return '天'
    elif '周转' in metric_name and '天' not in metric_name:
        return '次/年'
    elif '比率' in metric_name or '倍数' in metric_name:
        return '倍'
    else:
        return '元'
```

文件: `modules/metrics_loader.py`，行号 172-201

同时支持从配置文件加载单位：

```python
def get_unit(self, alias_or_field: str) -> str:
    """获取指标单位"""
    config = self.load_config()
    
    # 尝试从配置中查找
    for table_config in config.get('tables', {}).values():
        for field_name, field_config in table_config.get('fields', {}).items():
            aliases = field_config.get('aliases', [])
            if alias_or_field in aliases or alias_or_field == field_name:
                return field_config.get('unit', '元')
    
    # 默认单位推断
    if '率' in alias_or_field or 'ratio' in alias_or_field.lower():
        return '%'
    elif '天数' in alias_or_field:
        return '天'
    elif '倍数' in alias_or_field or '比率' in alias_or_field:
        return '倍'
    else:
        return '元'
```

### D.3 全年数据智能提示

文件: `server/routers/chat.py`，行号 543-570

当用户查询全年（跨多季度）的**比率类指标**时，系统会自动检测并显示提示信息：

```python
# === 检查是否需要提示默认平均值 ===
# 条件: 1.数据是按年分组(结果中没有quarter) 2.指标是比率类 3.问题中没有明确指明"平均"等词
try:
    has_ratio = False
    is_annual = True
    
    # 检查是否包含比率指标
    for r in results:
        m_name = r.get('metric_name', '')
        if any(x in m_name for x in ["率", "比", "burden", "margin"]):
            has_ratio = True
        
        # 检查是否包含季度信息
        q = r.get('quarter') or r.get('period_quarter')
        if q:
            is_annual = False
    
    # 检查问题关键词
    explicit_keywords = ["平均", "最大", "最小", "每季", "季度", "明细", "趋势", "detail", "avg", "max", "min"]
    has_explicit_intent = any(kw in question for kw in explicit_keywords)
    
    if has_ratio and is_annual and not has_explicit_intent:
        warning_msg = "💡 *系统提示: 用户没有明确提示比率指标是查询明细还是平均值、最大值等统计值，系统默认计算平均值；如果希望更精确查询，请给出明确提示*\n\n"
        yield send_content(warning_msg)
```

### D.4 处理逻辑说明

| 场景 | 用户问题 | 系统处理 | 输出 |
|------|----------|----------|------|
| 查询全年增值税税率 | "ABC公司2023年增值税税率" | 识别为比率指标 + 全年数据 | 显示提示 + 平均值 |

| 明确要求最大值 | "ABC公司2023年增值税税率最高多少" | 检测到"最大"关键词 | 返回最大值 |
