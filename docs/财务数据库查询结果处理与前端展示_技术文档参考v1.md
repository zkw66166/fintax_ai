## 一、系统架构概述

财务数据查询系统采用前后端分离架构，通过 SSE（Server-Sent Events）实现实时流式数据传输，支持文本内容与图表数据的混合推送。

### 1.1 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 后端 | FastAPI + Python | RESTful API 服务 |
| 数据库 | SQLite | financial.db 企业财务数据库 |
| 前端 | React + Chart.js | UI 渲染与图表展示 |
| 实时通信 | SSE (Server-Sent Events) | 服务端推送 |

### 1.2 数据流架构

```
用户问题
    │
    ▼
┌─────────────────────────────────────────────────┐
│  FastAPI /chat/stream 端点                      │
│  (server/routers/chat.py)                      │
└─────────────────────────────────────────────────┘
    │
    ├──▶ FinancialQuery.search() ──▶ financial.db
    │       │
    │       ├── 格式化方法: format_results()
    │       │   └── 返回 Markdown 表格/列表
    │       │
    │       └── 图表数据生成: 内存中构建 chartData
    │           └── yield send_event("chart", chartData)
    │
    ├──▶ SSE Event: "message" ──▶ 前端渲染文本
    │
    └──▶ SSE Event: "chart" ──▶ 前端 ChartRenderer
            │
            └── 图表组件渲染
```

```python
def search(self, question: str) -> Tuple[Optional[List[Dict]], Optional[Dict], str]:
    """
    主查询入口
    """
    # 1. 识别企业名称
    company = self.match_company(question)
    if not company:
        return None, None, "company_not_found"
    
    # 2. 提取时间范围
    time_range = self.extract_time_range(question)
    
    # 3. 识别指标
    metrics = self.extract_metrics(question)
    
    # 4. 执行查询 (Text-to-SQL)
    results = self.execute_query(company['id'], time_range, metrics, question)
    
    if not results:
        return None, company, "no_data"
    
    return results, company, "success"
```

### 3.1 格式化策略选择

文件: `modules/financial_query.py`，行号 1199-1223

```python
def format_results(self, results: List[Dict], company: Dict) -> str:
    """
    格式化查询结果
    当指标 > 1 或期间 >= 4 时使用表格格式
    """
    # 统计不同指标和期间
    metrics = set(r['metric_name'] for r in results)
    periods = set((r['year'], r.get('quarter')) for r in results)
    
    # 判断是否使用表格格式
    use_table = len(metrics) > 1 or len(periods) >= 4
    
    if use_table:
        return self._format_as_table(results, metrics, periods)
    else:
        return self._format_as_list(results)
```

```python
def _format_as_table(self, results, metrics, periods) -> str:

    # 按期间排序

    sorted_metrics = sorted(metrics)

    separator = "|------|"

        header += f" {metric} |"
        separator += "--------|"

        for metric in sorted_metrics:
            value = data_matrix.get((period formatted = self._, metric))
```

```python
def _format_value(self, value, unit: str) -> str:
    """格式化数值 - 自动单位转换"""
    if value is None:
        return "暂无数据"
    
    if unit == '元':
        if abs(value) >= 100000000:
            return f"{value/100000000:.2f}亿"
        elif abs(value) >= 10000:
            return f"{value/10000:.2f}万"
        else:
            return f"{value:.2f}"
    else:
        return f"{value:.2f}{unit}"

def _format_change(self, change, unit: str) -> str:
    """格式化增长额（带正负号）"""
    if unit == '元':
        if abs(change) >= 100000000:
            return f"{change/100000000:+.2f}亿"
        elif abs(change) >= 10000:
            return f"{change/10000:+.2f}万"
        else:
            return f"{change:+.2f}"
    else:
        return f"{change:+.2f}{unit}"
```

```python
        # 判断趋势
        if change_pct > 1:

        elif change_pct < -1:
            trend = 'down'
        else:

            'first_value': first_val,
            'last_value': last_val,
```

```javascript
const eventSource = new EventSource(`/api/chat/stream?question=${encodeURIComponent(question)}`);

    const data = JSON.parse(event.data);
```
