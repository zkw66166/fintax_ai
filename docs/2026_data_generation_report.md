## 数据生成方法

采用**复制-缩放法**（Copy-and-Scale Approach）：
1. 从2025年数据复制结构和模式
2. 应用各企业的增长率进行缩放
3. 保持数据的内部逻辑关系
4. 使用 `INSERT OR IGNORE` 确保不破坏原有数据

### 主要失败类型

1. **科目余额等式** (约192-144条失败)
   - 原因：简单缩放导致的小额差异
   - 影响：可接受的舍入误差

2. **增值税/企业所得税计算** (约48-24条失败)
   - 原因：税额计算与缩放后的收入不完全匹配
   - 影响：允许的差异范围内

3. **跨表校验** (约15条失败)
   - 原因：不同表之间的缩放因子略有差异
   - 影响：在容差范围内

4. **期间连续性** (约16-19条失败)
   - 原因：月度增长导致的期初期末差异
   - 影响：符合业务增长逻辑

### 发票数据差异

根据要求，**允许发票数据与申报数据之间存在一定差异**，这符合实际业务场景。
